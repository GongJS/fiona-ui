# API

## Breadcrumb

|    属性    |   说明   |    类型    |  默认值  | 是否必须 |
| --------- | ------- | --------- | ------- | ------- |
| children    |  |  ReactElement[]   |       | 是 |
| separator | 分隔符 | string | / | 否 |

## Breadcrumb.Item

|    属性    |   说明   |    类型    |  默认值  | 是否必须 |
| --------- | ------- | --------- | ------- | ------- |
| href    |  跳转的链接 |  string  |       | 否 |
| children    |  |  ReactElement[]  |       | 是 |
| separator    |  分隔符 |  string  |    /  | 否 |
